Запустить PowerShell от имени администратора.
Чтобы разрешить выполнение локальных подписанных скриптов либо всех доверенных сценариев 
необходимо изменить политику выполнения PowerShell.
В терминале запустите команду Set-ExecutionPolicy Bypass -Scope Process
Эта команда позволит запустить сценарий только в текущем сеансе PowerShell. После закрытия окна PowerShell настройки вернутся обратно.

Запуск теста 03_shop_test.py и формирование отчета для проверки функциональности интернет-магазина на сайте https://www.saucedemo.com/
Перейдите в директорию, где находится файл run_03_shop_test.ps1.
Выполните скрипт командой ".\run_03_shop_test.ps1"


Перед запуском теста test_form_01.py и формирование отчета для проверки функциональности
тестового сайта https://bonigarcia.dev/selenium-webdriver-java/data-types.html
Открыть в редакторе файл test_form_01.py,
Скачать драйвер для браузера Edge.
В файле test_form_01.py в строке вместо <Путь> подставить путь расположения драйвера
path: str = r"<Путь>" 
Перейдите в директорию, где находится файл run_test_form_01.ps1.
Выполните скрипт командой ".\run_test_form_01.ps1"
